# Cross-Lane Comparability Warning (Tool Evidence Completeness)

Date: 2026-05-12

## Warning
Cross-lane completeness statistics are not directly comparable as model quality claims.

## Reason
Observed completeness is a compound effect of:
- model
- harness
- reviewer pressure
- workflow shape
- prompt structure
- runtime hooks
- artifact visibility
- retry policy
- review sequencing
- token budget
- enforcement wording

## Allowed Use
- Within-lane trend monitoring over time.
- Reviewer process diagnostics.
- Observability gap discovery.

## Disallowed Use
- "Lane A governance better than Lane B" solely from completeness counts.
- Any completion-claim escalation using cross-lane completeness differences.
